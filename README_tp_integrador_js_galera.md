
# TP Integrador - JavaScript (Carrito de compras)

**Autor:** Lucia Galera  
**Archivo principal:** tp_integrador_js_galera.js

## Descripción
Proyecto de ejemplo que modela un carrito de compras. Cumple las consignas:
- Variables y tipos de datos (string, number, boolean, null)
- Objeto literal `Carrito` con métodos que usan `this`
- Array `catalogo` y recorrido con `forEach()`
- Función tradicional `buscarProductoPorId`
- Función flecha `crearItem`
- Control de flujo: `if/else` (implícito en algunas funciones), operador ternario, `switch`, `do...while`
- Demostración de `call` y `apply`

## Cómo ejecutar
- En un navegador: incluir el archivo en una página HTML con un `<script src="tp_integrador_js_galera.js"></script>` y abrir la consola.
- En Node.js:
  1. Colocar `tp_integrador_js_galera.js` en una carpeta.
  2. Ejecutar `node tp_integrador_js_galera.js`
